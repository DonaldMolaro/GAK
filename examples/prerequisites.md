# Prerequisites

This is a lightweight map of what helps before each stage of the `GAK` course.

## Before Starting

Useful background:

- basic C++ reading ability
- comfort with compiling and running command-line tools
- basic optimization intuition: better, worse, objective, constraint

Not required:

- prior genetic algorithm experience
- advanced math
- prior evolutionary-computing coursework

## Before Binary And Symbolic Examples

Helpful concepts:

- arrays or vectors
- booleans and finite-state values
- how an encoding maps to a problem

## Before Constraint Examples

Helpful concepts:

- constraints vs objectives
- exact solutions vs partial progress
- penalties and tradeoffs

## Before Constraint-Aware Sudoku

Helpful concepts:

- why generic edits can destroy useful structure
- why preserving invariants can reduce wasted search
- the difference between representation design and parameter tuning

## Before Writing A New Problem

You should be comfortable answering:

1. What does one gene mean?
2. How many genes are needed?
3. How many states can each gene take?
4. How do you score partial progress?
5. When is the problem solved exactly?
6. Do the generic operators preserve what matters?
